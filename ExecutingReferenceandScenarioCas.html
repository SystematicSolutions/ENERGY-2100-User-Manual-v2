Executing Reference and Scenario Cases

RunE2100.bat is a batch file that executes ENERGY 2100 scenarios given a set of input parameters. The required parameters in the current version of the batch are listed at the top of the file and include the scenario, start and end years for execution, type of outputs to produce, and the starting, base case, and reference databases. These parameters are passed to the model for execution through to the specified end year. This batch file is rarely modified directly, but instead produces different forecasts by altering the parameters sent to it.


RunModel.bat will usually include several RunE2100 calls that execute default forecast scenarios, such as for the base and reference cases. RunE2100 can also be executed individually to create a single scenario, or called in sequence from another batch file to queue multiple scenarios. RunE2100Runs.bat serves as a template for executing forecast scenarios outside of RunModel. The parameters in this file can be adjusted by the user as needed. The descriptions for each parameter are listed at the top of the file for user reference.


To execute a reference case, follow the instructions below:


  1. These instructions to execute the Reference Case assume you have already executed a RunAll.bat previously and have databases created for the Base run.
  2. To execute a reference case named Ref25, set up the batch file, RunE2100Runs.bat, to call RunE2100 and pass the arguments listed below:

                                    %1         %2       %3                %4        %5       %6       %7        %8       %9

Call   RunE2100    Ref25   Ref25   TestEmpty   2020    2050    Base     Base     Base    All


where:

Argument 1 (%1) - Scenario Name

Argument 2 (%2) - Reference Policy File

Argument 3 (%3) - Test Policy File

Argument 4 (%4) - Beginning Year

Argument 5 (%5) - Ending Year 

Argument 6 (%6) - Reference Case

Argument 7 (%7) - Scenario for zInitial in Access outputs

Argument 8 (%8) - Economic Model Investments Case (see Jeff)

Argument 9 (%9) - Output Files (All, ExcelDTAs, AccessDTAs, or None)
