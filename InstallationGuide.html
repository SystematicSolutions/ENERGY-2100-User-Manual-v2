Installation Guide

ENERGY 2100 requires installation of Julia, its dependencies, and 7zip. Optionally, you may also want to install Visual Studio Code (VS Code) to have the capability to write Julia code in an environment that recognizes the Julia language (and thereby adds color coding and catches syntax errors). 


Follow the instructions below to install each of these applications.


1.  Download ENERGY 2100 Model Files (See Obtain ENERGY 2100 Model)

  1. Create a subdirectory for the entire model directly on the computer’s main hard drive, such as 2100CanadaOak.
  2. Download the specified model files from SSI's ftp site to the subdirectory just created on your computer.
  3. Double-click the UnZipFilesForUpload.bat inside of the subdirectory. Double-clicking this file will unzip all of the model files and folders.


2.  Install Julia

    • Download the 64-bit Windows installer from www.julialang.org/downloads
    • Install Julia version (v1.10.3 April 30, 2024)
    • During installation, make sure to select "Add Julia to PATH"


3.  Install Julia Dependencies

    • To install dependencies required by Julia, please run batch file in ENERGY 2100 model files: \2020Model folder:  

double-click Install.com


4.  Verify 7-Zip is Installed on Computer

    • If 7-Zip is not installed, download and install from:  www.7-zip.org/download.html


5.  Install Visual Studio Code (VSCode)

    • Download the VS Code installer for you operating system from code.visualstudio.com
    • Run the installer and follow the setup prompts. Make sure you click on the "Open with Code" option.
    • Add Julia language extension

1. Click the Extensions icon in the Activity Bar

2. Search for an extension (e.g., "Julia")

3. Click "Install"
